# Chen Wen-chen Project

## Overview

Research project investigating the Chen Wen-chen (陳文成) incident using primary source documents.

## Source Documents

Located in `docs/pdfs/`:

1. `促轉會《陳文成案調查報告》001-072.pdf` - Transitional Justice Commission investigation report (pages 1-72)
2. `促轉會《陳文成案調查報告》073-170.pdf` - Transitional Justice Commission investigation report (pages 73-170)
3. `陳文成之死因.pdf` - Cause of death analysis
4. `陳文成事件調查報告-黃怡_林世煜.pdf` - Investigation report by Huang Yi & Lin Shi-yu
5. `Wecht-為民主而死（Death in the Cause of Democracy）.pdf` - Wecht forensic report
