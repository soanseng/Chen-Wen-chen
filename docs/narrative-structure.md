## 章節結構（7 章）

### 第一章：學者之路（1950–1978）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 1950–1978 |
| 敘事篇幅 | 短（序章性質） |
| 情緒基調 | 溫暖、希望、平靜 |
| 色調建議 | 暖色調，校園的金色光線 |

**敘事重點：**
- 陳文成出生於林口，家境清寒但天資聰穎
- 赴美留學，密西根大學統計學博士
- 受聘 Carnegie Mellon 助理教授，年僅 28 歲
- 與妻子陳素貞育有一子艾瑞克（Eric）

**關鍵段落：**
- 成長於二二八事件後的戒嚴台灣，但家庭「與政治素無淵源」
- 在匹茲堡的平靜學術生活——為後續悲劇建立反差

**主要來源：** Wecht Report p.1, 3；促轉會調查報告 p.1；黃怡、林世煜調查報告 p.3

### 第二章：海外的聲音（1978–1981）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 1978–1981 |
| 敘事篇幅 | 中 |
| 情緒基調 | 從熱忱轉向不安——暗潮湧動 |
| 色調建議 | 暖色漸冷，監控元素用冷色穿插 |

**敘事重點：**
- 《美麗島雜誌》與海外台灣人運動
- 陳文成為雜誌募款、尋找英譯者
- 「彩虹資料」——五筆跨太平洋電話監聽紀錄（1979.09–1981.05）
- 美麗島事件（1979.12.10）、林宅血案（1980.02.28）的時代背景
- 郵件攔截：連家書都不放過

**關鍵段落：**
- 「從電話監聽到郵件攔截，從被動監控到主動追蹤——一張綿密的網，正等待著陳文成歸來。」
- 陳錦華電話中提及「用長途電話不方便」——監控者意識到被監控

**敘事功能：** 建立「國家暴力的無所不在」，讓讀者理解陳文成返台的危險脈絡。

**主要來源：** Wecht Report p.7；促轉會調查報告 p.33–43；黃怡、林世煜調查報告 p.8

### 第三章：歸途（1981年5月—7月1日）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 1981.05.20 – 1981.07.01 |
| 敘事篇幅 | 中 |
| 情緒基調 | 表面平靜、暗流洶湧——陷阱正在收攏 |
| 色調建議 | 漸暗，壓迫感遞增 |

**敘事重點：**
- 返台探親，出境許可即被扣留（事實上被困在台灣）
- 師大「安定小組」線民佈建，監控升級
- 依然活躍：文化大學、台大、中研院演講
- **6月30日：第一次約談**——保安處偽裝為出入境管理局人員
- **7月1日：追問出境證未果**，航空公司有機位但無法出境
- 白健二晚餐證詞（1994 年回述，需標註可信度）

**關鍵段落：**
- 出境許可被扣——「他事實上被困在台灣，無法逃離情治機關的掌控」
- 「這是陳文成以自由之身度過的最後一個夜晚。」

**敘事功能：** 節奏收緊。從自由人變為被困者，為第四章的約談建立懸念。

**主要來源：** Wecht Report p.3, 7；促轉會調查報告 p.44–55；黃怡、林世煜調查報告 p.8–10

### 第四章：約談（1981年7月2日）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 1981.07.02（單日） |
| 敘事篇幅 | 長（全篇核心） |
| 情緒基調 | 壓迫、恐懼、空白——時間感凝滯 |
| 色調建議 | 最暗色調，審訊室的冷光 |

**三段式結構：**

1. **清晨——被帶走（05:00–08:30）**
   - 05:00 台大操場打籃球
   - 07:45 詢問妻子有無出入境管理局電話
   - 08:30 三名男子持約談傳票帶走（甯漢彤）
   - 「無預警」性質——事前未獲通知

2. **保安處——十二個小時（08:30–~21:30）**
   - 約談有錄音但非全程
   - 反覆施壓：承認台獨活動、交代在美關係
   - 出境許可作為交換條件
   - 陳文成自述：「自己本身有安全問題」
   - 跨文件矛盾：汪敬煦稱「心情很好」vs 筆錄的壓迫氣氛

3. **晚間——空白的六小時（~21:30–翌日07:00）**
   - 警總稱約九點半結束、派車送返
   - 五名目擊證人的證詞彼此矛盾
   - 促轉會結論：「無法證明警總確有將陳文成送返住處」
   - 鄧維祥家深夜訪問——「這可能是我們最後一次見面了」
   - 促轉會對鄧維祥證詞的質疑
   - 倪肇強證詞亦不可信
   - **核心結論：自7月2日早上被帶走後，現有證據無法證明陳文成曾脫離警總的實力支配**

**敘事功能：** 全篇最核心的章節。以逐時推進製造懸念，以跨文件矛盾揭示真相的複雜。

**主要來源：** Wecht Report p.3, 7–9, 13；促轉會調查報告 p.57–62；促轉會調查報告（下冊）p.64–87, 158–159；黃怡、林世煜調查報告 p.10–11

### 第五章：陳屍台大（1981年7月3日）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 1981.07.03 及後續法醫鑑定 |
| 敘事篇幅 | 最長（法醫證據密集） |
| 情緒基調 | 冷靜客觀的法醫語言——以克制呈現沉重 |
| 色調建議 | 冷灰色調，法醫報告的理性 |

**六段式結構：**

1. **清晨的發現**
   - 台大研究生圖書館建築描述（五層白磚水泥、消防太平梯、五樓護欄高 83cm）
   - 校警陳治國接不知名電話 → 劉仁富發現遺體
   - 遺體姿態：仰臥橫跨淺水溝

2. **遺體姿態的矛盾**
   - DeGroot 記載 vs 檢察官紀錄：雙臂下垂 vs 雙臂上舉如舉重

3. **衣著異常**
   - 皮帶繫於襯衫外、未穿襪子
   - 襯衫背部撕裂 37.5cm、褲襠後半部撕裂 41cm
   - 「彷彿有看不見的手能夠讓皺褶的衣服再塞回皮帶內」

4. **傷痕分析**
   - 體內傷清單（右側肋骨多處骨折、脊椎骨折、右腎裂傷等）
   - 體外傷清單（右背擦傷，但手部、腕部、指甲無傷）

5. **關鍵疑點**（六大疑點）
   - 傷勢方向與陳屍位置不符 → 屍體被移動
   - 無自我防禦傷痕
   - 落地後仍存活半小時以上
   - 可能的致昏手法（頭部打擊或麻醉）
   - 遺體如何進入校園？（深夜無車輛進出紀錄）
   - 五樓護欄事後重新上漆

6. **死因研判**
   - Wecht：最可能的死因是謀殺（homicide）
   - 促轉會：他殺可能性高
   - 所有接觸者均不認為陳文成會自殺

**敘事功能：** 以法醫證據的精確語言，讓事實自己說話。避免煽情，以克制的客觀達到最大衝擊。

**主要來源：** Wecht Report p.4–5, 9–12；促轉會調查報告（下冊）p.92–106, 159；陳文成之死因全文；黃怡、林世煜調查報告 p.31

### 第六章：追尋真相（1981–2020）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 1981.07 – 2020 |
| 敘事篇幅 | 長 |
| 情緒基調 | 憤怒與無力交織——系統性的真相壓制 |
| 色調建議 | 灰暗中偶有亮光（Wecht 調查、促轉會） |

**七段式結構：**

1. **定調——「畏罪自殺」**
   - 7/4 警總記者會（法醫調查未完即定調）
   - 7/5 否認曾約談（自相矛盾）
   - 7/19 檢察官偵查報告：「自殺或意外墜樓」

2. **Wecht 的獨立調查（1981.09）**
   - 三天行程：殯儀館驗屍、檢查衣物、警總會面、台大現場
   - 國安局全程監控
   - 松山機場記者會：「最可能的是謀殺」

3. **懲罰說真話的人**
   - 美聯社記者周清月因「驗屍」用詞被撤銷記者資格

4. **全面妨礙真相**
   - 監控家屬、查禁紀念文集
   - 三百餘封反制投書操縱國際輿論
   - 建議「從速訂定記者法」

5. **升官——嫌疑方主管變偵辦主導者**
   - 汪敬煦從警總總司令升任國安局局長

6. **秘密實驗——排除自殺（1984）**
   - 「明園專案」人體模型拋落實驗
   - 結果：排除自殺，但列為機密
   - 汪敬煦簽呈蔣經國：「自殺之可能性，應予排除」
   - 「本案政治性大於法律性」——真相被封印

7. **漫長的調查之路**
   - 七波調查比較表（1981–2020）
   - Wecht 1985 年論文發表
   - 1990 年陳寶月控告汪敬煦

**敘事功能：** 揭示威權體制如何系統性地壓制真相，從定調、監控、操縱輿論到人事安排，層層堆疊。

**主要來源：** Wecht Report p.2–13；促轉會調查報告 p.8, 13–19；促轉會調查報告（下冊）p.92–97, 115–156

### 第七章：未完的故事（2020—）

| 項目 | 內容 |
|------|------|
| 時間範圍 | 2020– |
| 敘事篇幅 | 短（收尾性質） |
| 情緒基調 | 沉重但開放——留給讀者的問題 |
| 色調建議 | 漸亮但不至明朗，帶有未解的灰 |

**敘事重點：**

1. **促轉會的結論（2020.04.15）**
   - 四大結論：監控、未脫離警總支配、他殺可能性高、警總妨礙真相
   - 「警總對於陳文成之死，涉有嫌疑，甚為明顯」

2. **真相為何難明**
   - 威權時期全面妨礙偵辦
   - 民主轉型後遲遲未啟動轉型正義 → 證據湮滅

3. **整體評價**
   - 「政權維穩高於真相發現之邏輯」
   - 關鍵檔案遺失、錄音帶下落不明、證人過世、那封信至今無人見過

**敘事功能：** 不給結論，而是留下問題。讓讀者帶著未解的重量離開。

**主要來源：** 促轉會調查報告（下冊）p.157–160

## 整體敘事弧線

```
Ch1        Ch2        Ch3        Ch4        Ch5        Ch6        Ch7
希望 ──→ 暗流 ──→ 收攏 ──→ 黑暗 ──→ 證據 ──→ 壓制 ──→ 未解
 ↑                              ↑                    ↑
 溫暖的起點              情緒最低點           憤怒的高峰
```

**節奏設計：**
- Ch1–2：緩慢鋪陳，建立人物與時代背景
- Ch3：加速，懸念遞增
- Ch4：最密集的時間單位（單日），逐時推進
- Ch5：節奏放慢為法醫式的冷靜分析
- Ch6：時間跨度最大（40年），但以事件驅動保持節奏
- Ch7：收束，留白
