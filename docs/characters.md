# 角色人物表

> 本文件列出遊戲中出現的人物，標註在各份文件中的出現位置、推理簿中需判斷的項目，以及虛構化處理建議。
>
> **處理原則**：
> - 🟢 實名：公眾人物或已為公開紀錄
> - 🟡 半虛構：保留姓氏或關係稱呼，改細節
> - 🔴 虛構化：完全改名，僅保留角色功能
>
> **敏感度說明**：陳文成本人及其直系家屬為本案核心，其遭遇已廣為人知且家屬長期推動真相追查，建議實名處理。加害體系人物因涉及法律責任與歷史評價，建議以職稱或虛構名處理。

## 一、受害者

### 陳文成 — 旅美學人
- **處理方式**：🟢 實名
- **背景**：39年生，建中→臺大→密西根大學博士→卡內基梅隆大學助理教授。妻陳素貞，育有一子。
- **出現文件**：幾乎所有文件
- **推理簿欄位**：無（他是被調查的對象，不是被判斷的對象）
- **設計要點**：
  - 陳文成在遊戲中「不說話」——玩家透過他人的紀錄和證詞認識他
  - 唯一的「聲音」是錄音譯文中殘存的幾句話
  - 他的人格透過家書、妻子公開聲明書、友人回憶浮現
  - 「他熱愛生命，在這世界上，他擁有值得活下去的一切」——陳素貞

- **處理方式**：🟡 建議以「妻子」或「陳太太」稱呼，避免過度暴露個人細節

- **設計要點**：遊戲中最有溫度的聲音。她的偵訊筆錄是冷靜的事實陳述，但每一句都帶著壓抑的悲痛。

## 三、加害體系——警總

### 「總司令」（原型：汪敬煦）
- **處理方式**：🔴 虛構化為「總司令」，不用真名
- **出現文件**：DOC-OFF-02（專案會議）、DOC-OFF-05（呈總統簽呈）、DOC-OFF-06（撥發工作費）、C-16（升遷矛盾）
- **推理簿欄位**：
  - 「總司令在案發後的角色？」——選項：不知情 / 事後掩蓋 / 事前知情
- **可確認**：案發後同年升任國安局局長；主導後續調查；撥發工作費
- **不可確認**：是否事前知情約談安排；是否下達特定指令

### 「第三組組長」（原型：鄒小韓）
- **處理方式**：🔴 虛構化為職稱
- **出現文件**：DOC-INT-01（約談筆錄——主問人）、DOC-WIT-05（答覆明園專案提問）
- **推理簿欄位**：
  - 「約談過程是否有異常？」——選項：正常約談 / 有施壓 / 有暴力
- **可確認**：負責7月2日約談的訊問工作；答覆護送人員身高167公分
- **不可確認**：約談全過程中發生了什麼（錄音帶已毀）；退伍後移居海外，無法聯繫

### 「記錄員」（原型：王文繽）
- **處理方式**：🔴 虛構化為職稱
- **出現文件**：DOC-INT-01（約談筆錄——製作者）、DOC-INT-03（協調辦理錄音帶事宜）
- **推理簿欄位**：無獨立欄位
- **可確認**：負責7月2日筆錄製作；涉及錄音帶最後處置

### 「護送員」（原型：王憶華）
- **處理方式**：🔴 虛構化為職稱
- **出現文件**：DOC-WIT-05（被描述為送陳文成返家之人）、DOC-WIT-02（五名目擊證人）
- **推理簿欄位**：
  - 「護送員是否真的將陳文成送回家？」——選項：是 / 否 / 無法確認
- **可確認**：身高約167公分；宣稱送至二樓
- **不可確認**：是否真的送達（身高與目擊證人不符→C-12）

### 「帶人的軍官」（原型：甯漢彤）
- **處理方式**：🔴 虛構化為職稱
- **出現文件**：DOC-INT-04（帶走陳文成的過程）
- **可確認**：承認「並未事先照會陳文成」；拒絕將公文書留給家屬

### 「保安處處長」（原型：郭學周）
- **處理方式**：🔴 虛構化為職稱
- **出現文件**：DOC-OFF-02（專案會議）
