# 矛盾與線索圖譜

> 本文件整理報告中所有重大矛盾點，作為遊戲核心機制——玩家標記可疑段落，系統自動連結矛盾並解鎖新文件。
>
> 矛盾分為兩類：
> - **🔓 可解決**：交叉比對可得出結論
> - **🔒 永遠無解**：證據已被銷毀/遺失，推理簿上留白

|--------|--------|
| 鄒小韓答覆（DOC-WIT-05）：「21時30分完成筆錄，約22時安抵羅斯福路住處」 | 70年法醫鑑定書（DOC-FOR-01）：死亡時間推定為7月3日凌晨4至6時 |

- **深層矛盾**：鄧維祥的證詞本身也充滿問題（→ C-05）

## 推理簿永遠填不滿的格子

以下問題在遊戲中**設計為永遠無法完全回答**：

1. **7月2日晚間至7月3日凌晨，陳文成究竟在哪裡？**（錄音帶已毀、鄧維祥證詞不可信）
2. **約談過程中究竟發生了什麼？**（僅有節錄譯文，原始錄音帶消失）
3. **是誰下令殺害陳文成？還是「失手」？**（無直接證據）
4. **切結書是否存在？內容為何？**（從未被找到）
5. **陳文成真正的死亡地點在哪裡？**（陳屍處非第一現場，但第一現場未知）

> 這些空白不是遊戲的缺陷，而是遊戲的主題：有些真相，在威權體制的系統性銷毀下，可能永遠無法被完整拼湊。推理簿上的空格，就是這個事件留給我們的傷口。
